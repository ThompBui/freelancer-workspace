# ThompBui

프리랜서·소규모 팀을 위한 **프로젝트 워크스페이스** — CRM, 칸반, 계약, 클라이언트 인프라, AI 검색.

[English](README.md) | [Tiếng Việt](README.vi.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | [Español](README.es.md) | [Português](README.pt.md) | 한국어

## 주요 기능

- 대시보드, 프로젝트, 칸반, 간트
- CRM, 받은편지함, 계약서
- 도메인 / VPS 갱신 관리
- 협업자 (Messenger / Zalo)
- AI 워크스페이스 검색
- 7개 언어 UI
- 다크 / 라이트 모드

## 빠른 시작

```bash
npm install
npm run dev
```

데모 데이터는 브라우저에 저장 (백엔드 미연동).
