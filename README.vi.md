# ThompBui

**Workspace quản lý dự án freelance & agency** — CRM, Kanban, hợp đồng, hạ tầng khách, tìm kiếm AI và theo dõi gia hạn trong một giao diện hiện đại.

[English](README.md) | Tiếng Việt | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | [Español](README.es.md) | [Português](README.pt.md) | [한국어](README.ko.md)

## Tính năng

- **Tổng quan** — doanh thu, dự án, cảnh báo chậm tiến độ, gia hạn hosting
- **Dự án** — Kanban kéo-thả, Gantt, chat nội bộ & khách
- **CRM** — khách hàng, công ty, liên hệ nhanh
- **Hộp thư & thông báo**
- **Hợp đồng** — mẫu từ Google Docs, in PDF
- **Hạ tầng khách** — tên miền, VPS, phí gia hạn
- **Tài khoản đã mua** — Figma, tool, subscription
- **Cộng tác viên** — Messenger / Zalo
- **Tìm kiếm AI** — hỏi đáp trên dữ liệu workspace
- **Đa ngôn ngữ** — EN, ZH, JA, ES, PT, KO, VI (UI); dữ liệu demo giữ tiếng Việt
- **Dark / Light mode**

## Chạy dự án

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

React 19 · Vite 6 · Tailwind CSS 4 · Lucide · Mammoth (DOCX)

Dữ liệu mẫu trong bộ nhớ trình duyệt (chưa có backend).
