# ThompBui

面向自由职业者与小团队的**项目工作区** — CRM、看板、合同、客户基础设施、AI 搜索与续费提醒。

[English](README.md) | [Tiếng Việt](README.vi.md) | 简体中文 | [日本語](README.ja.md) | [Español](README.es.md) | [Português](README.pt.md) | [한국어](README.ko.md)

## 功能

- 仪表盘、项目看板（Kanban）、甘特图
- 客户关系管理（CRM）
- 收件箱与通知
- 合同模板（支持 Google Docs / DOCX）
- 客户基础设施（域名、VPS、续费）
- 协作者与 Messenger / Zalo
- AI 工作区搜索（OpenAI 兼容 API）
- 界面语言：英语、简体中文、日语、西班牙语、葡萄牙语、韩语、越南语
- 深色 / 浅色主题

## 快速开始

```bash
npm install
npm run dev
```

## 技术栈

React 19 · Vite 6 · Tailwind CSS 4

演示数据保存在浏览器本地（暂无后端）。
