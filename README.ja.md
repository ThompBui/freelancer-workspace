# ThompBui

フリーランス・小規模チーム向けの**プロジェクトワークスペース** — CRM、カンバン、契約、クライアントインフラ、AI 検索。

[English](README.md) | [Tiếng Việt](README.vi.md) | [简体中文](README.zh-CN.md) | 日本語 | [Español](README.es.md) | [Português](README.pt.md) | [한국어](README.ko.md)

## 主な機能

- ダッシュボード、プロジェクト、カンバン、ガント
- CRM、受信トレイ、契約書
- ドメイン / VPS の更新管理
- コラボレーター（Messenger / Zalo）
- AI ワークスペース検索
- 7 言語 UI（EN / ZH / JA / ES / PT / KO / VI）
- ダーク / ライトモード

## セットアップ

```bash
npm install
npm run dev
```

デモデータはブラウザ内（バックエンド未接続）。
