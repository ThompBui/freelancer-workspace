# ThompBui

**Workspace para freelancers e agências** — CRM, Kanban, contratos, infraestrutura de clientes e busca com IA.

[English](README.md) | [Tiếng Việt](README.vi.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | [Español](README.es.md) | Português | [한국어](README.ko.md)

## Recursos

- Dashboard, projetos, Kanban, Gantt
- CRM, caixa de entrada, contratos
- Domínios, VPS, renovações
- Colaboradores (Messenger / Zalo)
- Busca IA (API compatível com OpenAI)
- Interface em 7 idiomas
- Tema escuro e claro

## Início rápido

```bash
npm install
npm run dev
```

Dados de demonstração no navegador (sem backend por enquanto).
