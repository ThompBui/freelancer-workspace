# ThompBui

**Espacio de trabajo para freelancers y agencias** — CRM, Kanban, contratos, infraestructura de clientes y búsqueda con IA.

[English](README.md) | [Tiếng Việt](README.vi.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | Español | [Português](README.pt.md) | [한국어](README.ko.md)

## Características

- Panel, proyectos, tablero Kanban, Gantt
- CRM, bandeja de entrada, contratos
- Dominios, hosting, renovaciones
- Colaboradores (Messenger / Zalo)
- Búsqueda IA (API compatible con OpenAI)
- UI en 7 idiomas
- Modo oscuro y claro

## Inicio rápido

```bash
npm install
npm run dev
```

Datos de demostración en el navegador (sin backend aún).
